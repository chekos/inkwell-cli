# Security Policy

## Supported Versions

Security fixes are released from the latest published version of Inkwell CLI.

## Reporting a Vulnerability

Please do not open public issues for suspected vulnerabilities. Use GitHub's private vulnerability reporting:

https://github.com/chekos/inkwell-cli/security/advisories/new

Include:

- A description of the vulnerability and impact
- Reproduction steps or proof of concept
- Affected version or commit, if known
- Any suggested mitigations

I will acknowledge valid reports as soon as practical and coordinate a fix before public disclosure.
